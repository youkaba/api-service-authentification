Pour accéder à votre dossier

Entrez le numéro de client (8 chiffres) et le mot de passe liés à votre dossier.

Message aux abonnés utilisant le navigateur Web Safari.

Il est possible que vous éprouviez certaines difficultés de navigation après vous être authentifié sur le portail de BAnQ. Nous vous recommandons d'utiliser les applications Firefox ou Google Chrome jusqu'à ce que ce problème soit définitivement réglé.

Merci de votre compréhension.

Comment s'abonner?

Pour vous inscrire à l'extranet, veuillez compléter ce formulaire d’inscription.

Vous recevrez votre numéro de client et mot de passe dans un délai maximum de trois jours ouvrables.
